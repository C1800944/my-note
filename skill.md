# JavaScript 基础练习 (js-basics-practice)

创建包含 JavaScript 基础知识问答的交互式 HTML 页面，用于学习和巩固 JS 核心概念。

## 工作流程

1. **创建目录** — 在项目目录下创建 `js-basic/` 文件夹
2. **创建 HTML 文件** — 在 `js-basic/` 目录下创建 `0415.html`
3. **编写问答功能** — 实现聊天界面和知识库匹配功能
4. **添加知识点** — 包含变量、数据类型、表达式、流程控制、函数等内容
5. **测试验证** — 浏览器打开文件，验证问答交互正常

## 行为规则

- 使用聊天界面展示问答内容
- 必须包含快捷问题按钮（变量、数据类型、函数、循环等）
- 知识库涵盖：变量定义、数据类型、表达式、流程控制、函数、数组、对象
- 支持关键词匹配，智能回复用户问题
- 所有作业提交到同一个仓库，不创建新仓库
- 提交信息格式：`添加JS基础练习skill·用户名/仓库名@commit哈希`

## 使用方式

当用户说「创建 JS 问答页面」或「JS 基础练习」时，直接生成完整的问答 HTML 文件并保存到 `js-basic/0415.html`。
