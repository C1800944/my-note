JavaScript 基础教程

0415.html 
<script>
    // 1. 变量声明
    var name = "Alice"; // 传统的变量声明方式
    console.log(name); // 输出: Alice
    let age = 30; // ES6 引入的块级作用域变量声明
    const PI = 3.14; // ES6 引入的常量声明

    // 2. 数据类型
    let str = "Hello, World!"; // 字符串
    let num = 42; // 数字
    let bool = true; // 布尔值
    let arr = [1, 2, 3]; // 数组
    let obj = { name: "Alice", age: 30 }; // 对象
    let undef; // undefined
    let nul = null; // null

    // 3. 函数定义
    function greet(name) {
        return "Hello, " + name + "!";
    }

    // 4. 条件语句
    if (age > 18) {
        console.log("You are an adult.");
    } else {
        console.log("You are a minor.");
    }

    // 5. 循环语句
    for (let i = 0; i < arr.length; i++) {
        console.log(arr[i]);
    }

    // 6. 表达式
    let sum = num + 10; // 算术表达式
    let isAdult = age > 18; // 逻辑表达式
    console.log(sum);
    console.log(isAdult);
</script>

代码说明

本示例展示了 JavaScript 的核心基础知识：

  • 变量声明：使用 var、let、const 声明变量
  • 数据类型：字符串、数字、布尔值、数组、对象、undefined、null
  • 函数定义：使用 function 关键字定义函数
  • 条件语句：if...else 进行条件判断
  • 循环语句：for 循环遍历数组
  • 表达式：算术表达式和逻辑表达式
