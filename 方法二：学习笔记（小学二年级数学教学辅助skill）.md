# 学习笔记（小学二年级数学教学辅助skill）

# 一、技能描述

本技能名为「学习笔记」，核心功能是辅助小学二年级数学教学，贴合二年级学生的认知特点和数学学习重点，覆盖100以内加减法、乘法口诀、认识人民币、长度单位换算、角的认识等核心知识点。技能以趣味化、通俗化的语言引导学生思考、练习，同时为教师/家长提供简单易懂的教学辅助思路，可配合课堂练习、课后巩固使用，帮助学生夯实数学基础，培养基本的数学思维，降低二年级数学学习难度，让学生在互动中提升数学学习兴趣。

适用人群：小学二年级学生、二年级数学教师、学生家长；使用场景：课堂互动、课后作业辅导、居家练习、知识点巩固。

# 二、触发词

触发词：/newslearn（输入该触发词后，技能立即启动，进入互动教学模式）

# 三、使用方法

1. 启动技能：在对话界面输入触发词「/newslearn」，发送后即可启动「学习笔记」技能，系统会自动发送欢迎语及功能引导。

2. 选择学习内容：启动后，系统会列出二年级数学核心知识点选项（100以内加减法、乘法口诀、认识人民币等），用户（学生/教师/家长）可直接回复对应知识点序号或名称，选择需要学习、练习的内容。

3. 互动学习/练习：根据选择的知识点，系统会先以通俗的语言讲解核心要点（结合生活实例），讲解完成后，发送练习题（由浅入深），用户回复答案后，系统会即时反馈对错，并给出详细解析，帮助理解错题原因。

4. 切换/结束技能：学习过程中，可直接回复「切换知识点」，系统会重新列出知识点选项；回复「结束学习」，即可退出技能，系统会发送学习总结及鼓励话语。
